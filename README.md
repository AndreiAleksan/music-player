
# Text-to-music player

## Description
This is a simple program that takes a file path, reads it, and plays whatever notes are stored inside (such as C, D, B and so on)
It also has a special character ("-") you can use if you wish for a little "rest" between notes.

## How to use
You must first create a text file and write a song with note characters (and the "rest" character). 
Then, you must input the path of the file and it will play your song!
